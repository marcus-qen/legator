Architecture

How it works — in 60 seconds

One binary. One install command. No agents, no dependencies, no Java, no Docker required.

You → Install probe (7MB static binary) on each server ↓ Probe → Connects home over WebSocket (TLS) → Collects inventory: OS, CPU, RAM, disk, packages, services, users → Sends heartbeat every 30 seconds ↓ Control Plane → Dashboard shows everything → You type a question in Chat — AI has live probe data → AI suggests a fix → you approve → probe executes → Every action logged with before/after state ↓ Policy Engine → Probe can only do what policy allows → Observe (read-only) → Diagnose → Remediate → Risky commands require human approval → AI never has root. Probe never reasons alone.
