Legator

Your servers. Your models. Your rules.

Legator is a self-hosted fleet management control plane. It deploys AI probes that talk to your infrastructure — but the AI never touches your servers directly. Brain and hands are separate.

Brain decides. Probe executes. Policy gates everything.

What it does

You install a 7MB binary on a server. It connects home over WebSocket. You can ask it questions, give it work, and get real answers back: inventory, health, drift, suspicious changes, failed services. If a change is risky, it stops and asks for approval.

Every action is logged with before/after state. The LLM never has root. The probe never reasons on its own. You decide autonomy level per host.

Quick start

# Build
git clone https://github.com/marcus-qen/legator && cd legator && make

# Run
./bin/control-plane

# Connect a probe
TOKEN=$(curl -sf -X POST http://localhost:8080/api/v1/tokens | jq -r .token)
curl -sSL http://localhost:8080/install.sh | sudo bash -s -- --server http://localhost:8080 --token "$TOKEN"
